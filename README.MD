# Tool to automate building/deploying treez services

# Pre-req
 JDK
 NPM
 MAVEN
 DOCKER
 DOCKER-COMPOSE
 
 
 # Usage
 ``` % ./treez.sh -h                                                                                                                                                 
USAGE:
-b build all services
-d deploy all services
-m  rebuild databases
-r  rebuild <SERVICE>: eg -r hints
-D  redeploy container for <SERVICE>: eg -D hints
```
